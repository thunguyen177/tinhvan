How to do this in Python

In Python, you can perform a hypothesis test for equal means using the library scipy

Here's a step-by-step guide on how to do it:

1.     Import the necessary libraries:

from scipy import stats

import numpy as np

2.     Collect your data for the old and new cookies. Let's assume you have two NumPy arrays, old_cookies and new_cookies, with the respective data:

old_cookies = np.array([20, 21, 22, 23, 22, 21, 20, 22, 23, 24])

new_cookies = np.array([25, 24, 25, 26, 25, 24, 23, 26, 27, 28])

3.     Set up your hypothesis test. You'll use the ttest_ind function from scipy.stats to perform an independent two-sample t-test, assuming equal variances:

t_stat, p_value = stats.ttest_ind(old_cookies, new_cookies, equal_var=True, alternative = 'less')

Note that here, equal_var is set to default (True). That means the function will perform a standard independent two-sample test that assumes equal population variances. However, if it is set to False, the function will perform Welchs t-test, which does not assume equal population variance.

Here, let us assume equal variance as the ingredients are similar.

Also, alternative has three options:

1/ two-sided: choose this when the alternative hypothesis is the means are unequal

2/ less: choose this when the alternative hypothesis is the mean of the first sample underlying distribution less than the mean of the mean of the second sample underlying distribution;

3/greater: choose this when the alternative hypothesis is the mean of the first sample underlying distribution is greater than the mean of the second sample underlying distribution.

So, we can also use the following codes to test the same hypothesis:

t_stat, p_value = stats.ttest_ind(new_cookies, old_cookies, equal_var=True, alternative = 'greater')

 

4.     Determine your significance level (alpha) to make a decision. For example, if you choose alpha = 0.05, it means you're willing to accept a 5% chance of making a Type I error.

alpha = 0.05

print('t_stat: ',t_stat)

print('p_value: ',p_value)

if p_value < alpha:

print("Reject the null hypothesis. The mean size of the new cookies is bigger than the mean size of the old cookies.")

else:

print("Fail to reject the null hypothesis. The new cookies are not significantly different from the old ones.")

The full codes, along with the result out is as follows,

from scipy import stats

import numpy as np

old_cookies = np.array([20, 21, 22, 23, 22, 21, 20, 22, 23, 24])

new_cookies = np.array([25, 24, 25, 26, 25, 24, 23, 26, 27, 28])

t_stat, p_value = stats.ttest_ind(old_cookies, new_cookies, equal_var=True, alternative = 'less')

alpha = 0.05

print('t_stat: ',t_stat)

print('p_value: ',p_value)

if p_value < alpha:

print("Reject the null hypothesis. The mean size of the new cookies is bigger than the mean size of the old cookies.")

else:

print("Fail to reject the null hypothesis. The new cookies are not significantly different from the old ones.")

How to do this in R

In R, you can perform a hypothesis test for equal means using the t.test function. Here's how to do it:

1.     Collect your data for the old and new cookies. You can create vectors or data frames with the respective data:

old_cookies <- c(20, 21, 22, 23, 22, 21, 20, 22, 23, 24)

new_cookies <- c(25, 24, 25, 26, 25, 24, 23, 26, 27, 28)

2.     Perform the t-test using the t.test function. Specify the data for the old and new cookies and indicate that you want a two-sample t-test assuming equal variances:

t.test(old_cookies, new_cookies, alternative = "less", var.equal = TRUE)

Note that here, var.equal is set to default (True). That means the function will perform a standard independent two sample test that assumes equal population variances. However, if it is set to False, the function will perform Welchs t-test, which does not assume equal population variance.

Also, alternative has three option:

1/ two.sided: choose this when the alternative hypothesis is the means are unequal

2/ less: choose this when the alternative hypothesis is the mean of the first sample underlying distribution less than the mean of the mean of the second sample underlying distribution;

3/greater: choose this when the alternative hypothesis is the mean of the first sample underlying distribution is greater than the mean of the second sample underlying distribution.

So, we can also use the following codes to test the same hypothesis

t.test(new_cookies, old_cookies, alternative = "greater", var.equal = TRUE)

 

3.     Pick alpha level

alpha <- 0.05

4.     Print the result to see the output, which includes the test statistic and p-value:

print(result)

if (result$p.value < alpha) {

cat("Reject the null hypothesis. The mean size of the new cookies is bigger than the mean size of the old cookies \n")

} else {

cat("Fail to reject the null hypothesis. The new cookies are not significantly different from the old ones.\n")

}

The whole codes is as follows,

old_cookies <- c(20, 21, 22, 23, 22, 21, 20, 22, 23, 24)

new_cookies <- c(25, 24, 25, 26, 25, 24, 23, 26, 27, 28)

result <- t.test(old_cookies, new_cookies, var.equal = TRUE)

alpha <- 0.05

print(result)

if (result$p.value < alpha) {

cat("Reject the null hypothesis. The mean size of the new cookies is bigger than the mean size of the old cookies.\n")

} else {

cat("Fail to reject the null hypothesis. The new cookies are not significantly different from the old ones.\n")

}
