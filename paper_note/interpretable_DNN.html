template matching

template matching in linear classifier

source: Lecture 2, Stanford's cs231n

When we stretch pixels into column, the cat's score is equal to the sum of the dot products of each line of the weight matrix times the pixels of the image plus a bias term term. Hence, linear classification can be seen as a template matching approach, where each of the row in the weight matrix corresponds to some template of the image, computing the dot product between each line of the weight matrix and the pixels gives us some kind of similarity this template for the class and the pixels of the image.
