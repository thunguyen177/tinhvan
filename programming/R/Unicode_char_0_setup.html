Error showed:

When this error is shown, we can not see the output file. However, there is a latex file with relevant name in the same directory. When I open the latex file to compile it in TexStudio instead, I found out the error: I copy and paste from some other page, and forgot to rewrite the notation \(\forall\) into math formula that latex can render. I fixed it by changing from "\(\forall\)" to "$\forall$" and then it works.
